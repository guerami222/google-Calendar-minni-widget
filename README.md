# Google Calendar Widget

간단한 데스크톱용 구글 캘린더 위젯

<img width="1601" height="982" alt="image" src="https://github.com/user-attachments/assets/036fa183-bee4-4e1d-ae9a-331cef40ae7b" />


## 기능
- Google Calendar 연동
- 수동/자동 동기화 
- 일정 표시
- 일정 추가/수정/삭제
- 항상 위 표시
- 투명도 조절
- 마지막 위치/사이즈 기억

## 사용 방법
1. npm install
2. npm start

## 기타
Electron 기반으로 제작됨
